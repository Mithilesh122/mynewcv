Newsletter-Signup Project

This Node.js app collects user info and adds it to a Mailchimp list using their API. Express framework creates a server that listens on port 3000. A GET request to the root route responds with a form, while a POST request with form data processes the data and adds the user to the mailing list. The app includes error handling and hard-coded Mailchimp API endpoint and key that need to be updated for a different account or list.

Weather Project

This is a Node.js web application that retrieves current weather information from the OpenWeatherMap API based on user input of a city name. The application uses Express for server-side routing and HTTPS module for making API requests. It also includes a simple HTML form for user input and dynamically displays weather data on the page using JavaScript.
