Firebase Movie Streaming App

This Android Studio project is a video streaming app that uses Firebase to store and stream movies. Users can select from a list of movies with thumbnail posters, and the video player has standard controls and buffering/loading indicators. An AsyncTask updates the player's progress and buffering. The app is a great example of using video strea ming and Firebase for Android development

Student Database Project

This is a simple Android app that lets users manage student records in a SQLite database. The UI has text fields for USN, name, phone, and address, as well as buttons for CRUD operations: insert, delete, update, and view all. There's also an SMS button to send a student's Marks to their phone number. The code uses SQL commands to interact with the database. Overall, this app provides an example of how to implement a database in an Android app and perform basic operations on it.
